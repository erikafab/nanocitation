Please insert the ID of a Nanopublication, which can be in the following forms:

  • The full Nanopublication URI (e.g. http://rdf.disgenet.org/resource/nanopub/NP526.RAoYM3ySzlMrorcFMKOep89Mqcwyp4CTLvegQb0syCBX4)
  • The portion of the Nanopublication URI which follows the last '/' character (e.g. NP526.RAoYM3ySzlMrorcFMKOep89Mqcwyp4CTLvegQb0syCBX4)
  • The portion of the Nanopublication URI which follows the last '.' character (e.g. RAoYM3ySzlMrorcFMKOep89Mqcwyp4CTLvegQb0syCBX4)
For now, only DisGeNET Nanopublications are supported.

Once you submit the request, you will get redirected to the landing page of the chosen Nanopublication, where you can find the text snippet to be copied and pasted in your reference list, and details about the Nanopublication.
