API Reference

You can use the Nanocitatation API to look up the metadata of nanopublications and to get their text-snippet to be used in the citation reference list. All Nanocitation REST API uses HTTP Protocol and all services provided are GET HTTP methods returning JSON, XML or TXT response. No authentication to access the API is requested.

Available method:



Get the metadata in XML/JSON of the nanopublication with id {id}
The media response type is XML by default, a JSON response can be requested by specifying it in the "Accept" header of the HTTP request or by affix the file extension in the request (as you can see in the below examples).
GET
/rest/meta/{id}



Get the text snippet of the nanopublication with id {id}

GET
/rest/textsnippet/{id}
Doc

Request Examples

Get the metadata in XML format of the nanopublication with id RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s from cURL request:

Curl
curl http://nanocitation.dei.unipd.it/rest/meta/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s
or

Curl
curl http://nanocitation.dei.unipd.it/rest/meta/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s.xml
or

Curl
curl http://ims-ws.dei.unipd.it/nanocitation/rest/meta/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s -H "Accept: application/xml"

Get the metadata in JSON format of the nanopublication with id RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s from cURL request:

Curl
curl http://nanocitation.dei.unipd.it/rest/meta/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s.json
or

Curl
curl http://nanocitation.dei.unipd.it/rest/meta/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s -H "Accept: application/json"



Get the metadata in text snippet of the nanopublication with id RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s from simple HTTP request:

HTTP
HTTP GET http://nanocitation.dei.unipd.it/rest/textsnippet/RA2SzT2Q5GbcpOKF1d2iZJAUsu0EagaTTPxiZsUSPwa0s
